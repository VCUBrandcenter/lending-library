How This Works

Simply look for an item that you would like to use for your project and if it's available you'll be asked to fill out a form to borrow it. Then see one of the CT professors to grab it for you.

If the item that you need is checked out, feel free to email the student that has borrowed it and ask when it might be checked back in. DO NOT BORROW FROM STUDENTS. Have the item returned back to the lab and then borrow it via the form again.

If you have any questions feel free to email us here.

Help Make This App Better

The site is built on an open source project, uses a Google Spreadsheet for the data and is designed to be duplicated and live in minutes (Fork-n-go!).

You can help us have a better tool for checking in and out items from the VCU Brandcenter Lab by forking this project on Github and make your changes, send us a pull request and we will add your tweaks.
